<Hide 'Run As Administrator'> button hides/shows 'Run as administrator'
entry in the right-click Explorer context menu. 

After hiding this entry, the user cannot force the file execution with 
Administrative Rights. Furthermore, the 'Command Prompt (Administrator)'
and 'Windows PowerShell Admin' options in Windows Power Menu are not 
available. The 'Run as administrator' entry in the Search context menu,
is not available too. 

See 'Hard_Configurator - Manual.pdf' for more help.
